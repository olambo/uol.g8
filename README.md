uol.g8: simple zio sbt project template
=================

A [Giter8][g8] template for a minimal zio project.

Template usage
--------------
Using `sbt`, do:
```
sbt new olambo/uol.g8

or locally, install g8:
g8 file://uol.g8
```
in the folder where you want to clone the template.
